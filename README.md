# 🚀 Django Customer & Order Management System

## 📌 Project Overview
This is a Django-based web application designed to manage customers, products, and orders efficiently.  
The system includes authentication, dashboard analytics, and full CRUD functionality for business management.

---

## ✨ Features

- 🔐 User Authentication (Login & Registration)
- 📊 Dashboard with Order Status Overview
- 👤 Customer Management (Create, Update, Delete)
- 📦 Product Management
- 📝 Order Creation & Tracking
- 🏷 Tag Management
- 🎨 Responsive UI with HTML & CSS
- 🗃 SQLite Database Integration

---

## 🛠 Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
